NERONE N9 — personal brand images
==================================
Drop these exact files into THIS folder
(assets/img/projects/nerone-n9/):

  logo.png          -> main logo (the one on the white background)
  business-card.jpg -> business card mockup (front + back)
  letterhead.jpg    -> letterhead mockup
  signage.jpg       -> building / studio sign mockup
  mug.jpg           -> branded mug (merch) mockup

Notes:
- logo.png is also used on the homepage hero and in the header badge.
  A transparent-background PNG looks best, but the white-background
  version works too (it sits on a clean white "brand plate").
- Extra logo variants you send can be added to the gallery later.
- Filenames must match exactly (lowercase). Once added, the site shows
  them automatically — no other edits needed.
